# Product Backlog

This file tracks features and enhancements planned for future release cycles beyond the Beta.

## Phase: Cyber-Organic Integration (Post-Beta)

### 1. Terminal-to-Cloud Sync (v2)
- [ ] **Real-time Collaboration:** Implement the ability to view and interact with shared logs or task lists in real-time.
- [ ] **Sync Snapshots:** Add version control for user data, allowing rollbacks to previous versions of notes or task lists from the cloud.

### 2. Extended Command Palette
- [ ] **Quick Macros:** Allow users to define custom text snippets or links that can be triggered directly via the command palette.
- [ ] **Integrated System Utilities:** Add a calculator and a timezone converter directly into the `kbar` command palette.

### 3. Visual Overhaul: CRT Realism
- [ ] **Signal Distortion:** Add optional "Glass Distortion" and "Chromatic Aberration" toggles for enhanced immersion.
- [ ] **Interference Animations:** Implement "Signal Interference" visual effects that trigger during data synchronization or deletions.

### 4. Mobile 'Pocket-Deck' View
- [ ] **Vertical Optimization:** Optimize the Retro UI for ultra-compact vertical screens.
- [ ] **Touch-Friendly Interactions:** Adapt navigation and editing features for mobile users while maintaining the brutalist aesthetic.
